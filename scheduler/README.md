# scheduler

## Purpose
Hosts scheduling domain specifications, trigger policies, and execution cadence documentation.

## Responsibilities
- Define trigger types, blackout windows, and retry policies.
- Maintain run orchestration contracts with queue domain.
- Ensure deterministic scheduling behavior.
