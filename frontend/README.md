# frontend

## Purpose
Enterprise React application for TKK Universal Validator user workflows and dashboards.

## Prompt 21 Implementation
- React + TypeScript + Vite foundation.
- Material UI dashboard shell with modern visual system.
- Dark and light themes with runtime toggle.
- Responsive desktop/mobile layout with adaptive drawer.
- Role-based navigation and route protection.

## Local Development
1. npm install
2. npm run dev
3. Open http://localhost:5173

## Build
- npm run build
- npm run preview
